# Self-Hosted Services

## Immich

[Immich](https://immich.app/) is a Google Photos alternative. It is used to store and manage my photos.

## Paperless-ngx

[Paperless-ngx](https://docs.paperless-ngx.com/) is a document storage and indexing solution. It is used to store and index digital copies of my documents like receipts, invoices, medical, insurance, etc.

## Jellyfin

[Jellyfin](https://jellyfin.org/) is a media streaming server. It is used to stream TV shows and movies in my collection.

## Seafile

[Seafile](https://www.seafile.com/en/home/) is a file syncing service, similar to Google Drive and OneDrive. It is used to sync my files across machines.

## Coder

[Coder](https://coder.com/) is a vscode-server alternative that builds ephemeral environments/workspaces for development within VS Code. It is used to provide me with reusable development environments. My main environment contains dependencies like Ansible, ansible-lint, python, PowerShell, Terraform, kubectl, and more. These environments are built in the backend using Terraform, so they can be destroyed and re-created easily.

## Home Assistant

[Home Assistant](https://www.home-assistant.io/) is a smart-home and automation platform. It is used to integrate with some of my smart home devices. HA provides a central platform for dashboards, device control, and home automation.

## Outline Wiki

[Outline](https://www.getoutline.com/) is a modern wiki platform, similar to Confluence. It is used as my central knowledge-base for documentation.
